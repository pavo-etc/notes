A Linux-based operating system with a pure Python userspace

snakeware is an experimental OS with developed in 2020. Its userspace is written entirely in Python, barring a single Busybox root process. It was created by Joshua Moore drawing inspiration from the Commodore 64. The window manager, snakewm, is written primarily with pygame_gui.
